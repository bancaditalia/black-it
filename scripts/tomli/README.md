# tomli 2.2.1

This is a vendorized version of tomli v2.2.1, taken from:
https://github.com/hukkin/tomli/commit/73c3d102eb81fe0d2b87f905df4f740f8878d8da

tomli is the upstream project from which the tomllib library in python 3.11
stdlib was forked.

Once the project start dropping support for python 3.10, this shim can be
deleted.
